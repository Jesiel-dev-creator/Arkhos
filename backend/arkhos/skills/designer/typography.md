# Typography System

## Type Scale

Use a consistent, mathematically derived scale for all text sizes. The recommended scale in pixels: 12 (caption), 14 (small/label), 16 (body), 20 (large body/lead), 24 (H4), 32 (H3), 48 (H2), 64 (H1 desktop), 80 (display/hero). On mobile, reduce the top end: H1 becomes 36-40px, H2 becomes 32px, display becomes 48px. Never use arbitrary sizes outside this scale.

## Font Pairing Rules

Pair one display/heading font with one body font — two typefaces maximum per page. The display font carries personality (bold, distinctive); the body font carries readability (clean, neutral). Contrast in style creates hierarchy: a geometric display font (Syne, Space Grotesk) pairs well with a humanist body font (DM Sans, Outfit). A serif display (Cormorant, Playfair) pairs well with a clean sans-serif body. Never pair two fonts that are too similar — the difference should be immediately visible.

## Heading Styles

Headings use bold (700) or extrabold (800) weight with tight letter-spacing (-0.02em to -0.04em for large sizes). Line height for headings: 1.1 to 1.2 (tight). Color: primary text color (frost/white on dark, void/black on light). Hero headings may use the accent color for one keyword to draw attention. Never fully capitalize headings longer than 3 words — it reduces readability.

## Body Text Styles

Body text at 16-18px, regular weight (400), with relaxed line height (1.5 to 1.65). Paragraph max-width: 65-75 characters per line for optimal readability. Color: slightly muted compared to headings (90% opacity or a dedicated muted color). Links: accent color with underline on hover, not permanently underlined. Bold within body text uses medium (500) or semibold (600), never the same weight as headings.

## Code Typography

Monospace font at 14px with regular weight (400). Line height: 1.6 for readability. Background: slightly elevated from the page background. Border-radius: 4-6px for inline code, 8px for code blocks. Syntax highlighting with a restrained palette that matches the page's color scheme.
