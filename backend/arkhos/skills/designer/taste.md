# Design Taste — Anti-Patterns and Pro Patterns

## Anti-Patterns (Never Do These)

- Purple-to-blue gradients on everything — the hallmark of generic AI-generated design
- Blob shapes and abstract SVG decorations with no purpose
- Comic Sans, Papyrus, or any novelty typeface
- Drop shadows on every element — creates visual clutter and feels dated
- Overcrowded hero sections with headline, subheadline, 3 CTAs, feature list, and video all above the fold
- Generic stock photos of people pointing at whiteboards or shaking hands
- Rainbow color palettes with 6+ accent colors competing for attention
- Inconsistent border radius (mixing sharp corners and rounded on the same page)
- Text over busy images without sufficient overlay or contrast
- Centered text blocks wider than 600px — unreadable
- Decorative animations that serve no functional purpose and slow the page
- Using more than 2 typefaces on a single page

## Pro Patterns (Always Follow)

- Strong typographic hierarchy: clear visual difference between H1, H2, body, and captions
- Grid-based layouts with consistent column structure (12-column grid preferred)
- Generous whitespace — when in doubt, add more space, not more content
- One accent color used consistently for interactive elements and emphasis
- Subtle, purposeful animations: 200-400ms transitions on hover, scroll reveals with restraint
- Consistent spacing scale derived from a base unit: 4, 8, 16, 24, 32, 48, 64px
- Photography with consistent tone and color grading across the page
- Card components with uniform padding, border radius, and shadow depth
- Maximum content width of 1280px with comfortable side margins
- Responsive breakpoints at 640, 768, 1024, and 1280px
